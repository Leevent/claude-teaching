FAQ

常見問題

我是 Claude 初學者，適合來嗎？ ▾
非常適合！小聚的參加者約 40% 是入門者。你可以看到別人的真實工作流，這比任何教學影片都有啟發。帶著你的筆電和問題來就對了。
需要自備什麼？ ▾
建議攜帶筆電和手機。現場會有 WiFi 和電源。如果想跟著操作或展示自己的工作流，筆電必備。
活動會錄影嗎？ ▾
不錄影，僅拍照作為活動紀錄。這是為了營造輕鬆自在的交流氛圍，讓大家放心分享。
和 Claude 線上講座有什麼不同？ ▾
線上講座（A 班、B 班）是系統化的課程教學，有完整教案和作業。小聚是社群交流活動，重點在面對面互動、經驗分享和人脈建立。兩者互補，建議都參加。
之後還會辦嗎？ ▾
會！目標是建立台中 Claude 社群，定期舉辦（至少每季一場，目標月聚）。首場已成功驗證需求，後續會邀請更多講者分享。
場地在哪裡？怎麼到？ ▾
台中涵軒雅敘共享中心。報名後會收到詳細地址、停車資訊和交通指南。建議提早 10 分鐘到場，可以先認識其他參加者。
我已經用 Claude 很熟了，還值得來嗎？ ▾
絕對值得。首場有 40% 是進階用戶，不少人用 Claude Code 做自動化。小聚最大的價值不是「學」，是看到別人怎麼把 Claude 用在你沒想過的場景。而且進階用戶的分享對新手幫助最大，你的參與本身就是社群價值。
我不是工程師，展示的 Claude Code 我看得懂嗎？ ▾
看得懂！展示的重點是「這能幫你做什麼」而不是「程式碼怎麼寫」。首場有花店老闆、行銷企劃、設計師，大家都看得很投入。Claude Code 的操作其實就是打字下指令，跟用 ChatGPT 差不多直覺。
可以帶同事 / 朋友一起來嗎？ ▾
歡迎！3 人以上團報可以私訊洽詢優惠。帶團隊一起來最棒的地方是：回去之後有人可以一起討論怎麼實際導入工作。聯繫方式：vent@leevent.co
